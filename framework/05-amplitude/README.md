# Estágio 5 — Amplitude (Specialization, Influence, Public Output)

## Por que esse estágio existe

O estágio 4 (Sistemas) já te dá um generalista forte: arquiteta sistemas distribuídos, lê DDIA, escreve ADRs, lidera projeto. Esse estágio cobre um eixo distinto que vem depois: especialização declarada, output público, mentoria, leitura de papers, contribuição em OSS, e capacidade de articular trade-offs em termos de negócio.

A diferença é qualitativa: aqui você **multiplica via influência** (não só via código), **especializa** (ou se posiciona como T-shape forte), **publica** (audiência além do empregador), **mentora** (outros crescem por você), **lê paper** (não só blog), **constrói from scratch** (entende abaixo da linha de abstração), **fala org** (Conway, team topologies), **pensa business** (unit economics), e **escolhe trajetória** (specialist deep ou generalist com peso).

Este estágio é o **gap closer**. Não é "mais um capstone"; é a integração + posicionamento de tudo o que veio antes, com módulos que cobrem dimensões fora do que código sozinho ensina.

**Promessa de saída:** você consegue articular evidência multidimensional — código, escrita pública, mentoria, comunidade, papers, e um eixo de profundidade declarado.

## Postura recomendada para este estágio

- **Specialization é escolha ativa**. Default = generalist; declare specialist se deliberada.
- **Public output não é vanity**. Write-to-think + audience compounds.
- **Mentoria não é favor — é prática**. Time budget allocated, disciplined.
- **Paper reading sustainable**. 1/semana, 12 semanas mínimo pra forma habit.
- **Promo case é currículo do trabalho técnico**. Document evidence à medida que produz.
- **Multi-domain capstones forçam transferência**. Stay out of comfort zone Logística.
- **Conway's Law é alavanca**. Org design importa tanto quanto code.
- **Build-from-scratch demystifica**. Construa pelo menos 2 dos 5 toys.

## Sobre cobiça e sobre cuidado

Esse estágio é sobre **práticas reais**, não tier de status. Se você faz só pra "colecionar título", o output será raso. Se você faz pra desenvolver-se nessas dimensões (clarity via writing, multiplicação via mentoria, depth via build-from-scratch), eventuais títulos são consequência.

Igualmente: nem todo mundo precisa fazer este estágio. IC consolidado no estágio 4 com vida cheia fora do trabalho é trajetória válida. Este estágio demanda investment alto. Decida com sobriedade.

---

## Tempo? Não — outra vez

Como demais estágios: **leva o que precisar**. Mas atenção: 6 módulos núcleo + capstone é 12-24 meses de trabalho real, depois de estágio 4 completo. Esse é o estágio mais longo. Comprometa-se com cadência, não calendário.
